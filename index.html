GlowCloser

A wearable that uses glowing lights as a form of real-time, nonverbal communication.

Check it out on GitHub!

Problem Statement

Real-time communication in long distance relationships can be difficult due to the need to coordinate a time and a particular medium. Simple or basic interactions are not always possible. Furthermore, video conferencing mediums like Hangouts or Skype are not always intuitive and easy to use for children. GlowCloser presents an Internet of Things approach to this problem using paired necklaces. By interacting with the necklaces, users can send a series of light movements to the paired bracelet. The device allows for simple, real-time communication for any two users around the world. GlowCloser can be used by romantic partners, parents, children, friends, or any two people are separated by time and space.

Design Process

Initial Concept

GlowCloser was initially conceived as a bracelet with glowing lights. Pictured above is our low fidelity prototype.

Technology Chosen

We picked the Particle Internet Button, which is designed for developing IoT devices by using a microcontroller with wifi capabilities.

Adaptation to Necklace

Based on the size limitations of the technology chosen, we decided to adapt the device to a necklace.

Conceptual Design

Initial concept

The system functions as a pair (or set) of simple necklaces. When User A interacts with his/her necklace, User B’s corresponding necklace will glow and light up based on the specific command/action User A used to stimulate the message.
Currently, there are three different customizable “messages”.

Tangible interactions (necklace --> app):
Rotating necklace --> Rotating lights
Shaking necklace --> Rainbow lights
Rotating lights <-- Rotating knob

Future iterations

In the next iterations of GlowCloser, the different “messages” can be represented by different colors and generated warmth from the necklace, using touch sensors. For example, User A can say “goodnight” to User B by holding down the sensor on his/her wristband, resulting in User B’s wristband to glow blue until User A releases the touch.

The tangible elements of this design include the physical necklace (the Internet Button), an orientation sensor, an acceleration sensor and the LED lights in the button. A tangible approach is the most beneficial for addressing this long-distance communication problem since it introduces a form of palpable remote communication.
